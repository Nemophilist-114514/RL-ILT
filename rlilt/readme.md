# RL ILT

This is the code of RL-ILT: Pixel-Based Reinforcement Learning for Inverse Lithography Technology, a pixel-based deep reinforcement learning framework for mask optimization.

## Installation

Install LithoBench : <https://github.com/shelljane/lithobench>

Install Dependencies :

```
pip install tqdm gymnasium
```

Move files :

```
mv /path_to_rlilt/rlilt/ /path_to_lithobench/lithobench/ilt/
```

## Run RL ILT

```
python lithobench/ilt/rlilt/main.py
```
